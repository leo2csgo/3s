# 3秒出卡 - 智能行程规划小程序

一款基于微信小程序云开发的智能旅行行程规划工具。用户只需输入**城市、天数和出行目的**，即可在3秒内生成精美的行程卡片图，适合社交分享。

## 核心功能

- **快速生成**: 3秒内生成个性化旅行行程卡片
- **智能匹配**: 基于规则引擎自动组合最优行程
- **精美卡片**: 生成适合社交分享的行程卡片图
- **多场景支持**: 亲子遛娃、情侣约会、朋友小聚、美食探店等多种出行目的

## 技术栈

- **前端**: 微信小程序原生开发 (WXML, WXSS, JavaScript)
- **后端**: 微信云开发
  - 云数据库 (存储活动素材)
  - 云函数 (规则引擎 + 图片生成)
  - 云存储 (背景图 + 生成的卡片图)

## 云数据库结构

### activities 集合
- `name`: 活动名称
- `city`: 所属城市
- `district`: 所属区县
- `tags`: 标签数组 (亲子/情侣/美食等)
- `category`: 分类
- `duration`: 建议游玩时长 (小时)
- `cost`: 预估人均费用 (元)
- `address`: 详细地址
- `description`: 一句话亮点描述

## 开发说明

1. 使用微信开发者工具打开项目
2. 开通并配置云开发环境
3. 创建云数据库集合 `activities`
4. 录入测试数据 (至少20条上海活动)
5. 上传云函数 `generateCard`
6. 编译运行小程序
